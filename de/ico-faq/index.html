Wann beginnt und endet der ICO?

Der ICO beginnt am 1. Oktober 2017 um 14:00 Uhr GMT. Sie endet am 1. November 2017, 14:00 Uhr GMT oder wenn die Obergrenze erreicht ist.
Bitte, senden Sie keine ETH an unseren Vertrag vor dem Start oder nach Ende des ICO (1. Oktober 2017, 14:00 Uhr GMT)! Die Transaktion wird zurückkommen, aber Sie werden dennoch Transaktionsgebühren der ETH bezahlen müssen.

Wie viele Dentacoin (DCN) werde ich für ein 1 ETH bekommen?

Der ICO Preis von Dentacoin wird bei 0,00011723 USD/DCN liegen. Der Preis in ETH wird eine Woche vor der ICO veröffentlicht.
Beispiel: Falls dann 1 ETH = 300 USD ist, dann werden Sie mit 1 ETH zirka 2.560.000 DCN (300/0,00011723) kaufen können.

Welches Wallet sollte ich für den Kauf von Dentacoin nutzen?

Als Ethereum-basiertes ERC20-Token kann DCN unter Verwendung bestehender Ethereum-Clients gespeichert und verwaltet werden, darunter MyEtherWallet.com, MetaMask, Mist, das offizielle Ethereum-Wallet.

Kann ich DCN mit einer FIAT-Währung kaufen (z. B. USD, EUR, INR)?

Während unseres Token-Verkaufs können Sie Dentacoin durch ETH direkt mit unserem Smart Contract kaufen. Wenn Sie keine ETH haben, können Sie diese auf einer Austauschplattform (Bitpanda, Coinbase, Kraken, Poloniex, Bittrex, Ether Rush – India, etc.) gegen Ihre bevorzugte FIAT-Währung tauschen.
VORSICHT: Denken Sie daran, dass der Prüfprozess auf einigen dieser Plattformen seit kurzem länger dauert als üblich (ein bis zwei Wochen), weshalb Sie sich vielleicht beeilen sollten, wenn Sie neu sind.

Kann ich mir meinen Platz beim ICO reservieren?

Nein. Wer zuerst kommt, mahlt zuerst. Sie können nur sicherstellen, dass Sie daran erinnert werden, indem Sie Ihr Interesse am ICO hier angeben: www.dentacoin.com/ico.

Gibt es ein Minimum oder Maximum von DCN, welches ich während des ICO kaufen muss?

Das Limit für den Minimalkauf liegt bei 10 DCN. Es gibt keine maximale Menge (es sei denn, es übersteigt die ICO-Obergrenze) - Sie können in genau dem Maße kaufen, in dem Sie unsere Mission unterstützen möchten: Die Zahnpflege weltweit zu verbessern und sie erschwinglich zu machen.

Wie viel Gas sollte ich nutzen, um sicherzugehen, dass meine Transaktion durchgeht?

Wir empfehlen das Gaslimit auf 200.000 zu setzen.

Welches Wallet sollte ich für den Beitrag benutzen?

Sie können jedes Wallet verwenden, für das Sie Ihren privaten Schlüssel haben. Allerdings ist es vorzuziehen, ein Wallet zu verwenden, das es Ihnen ermöglicht, benutzerdefinierte Token hinzuzufügen, um später Transfers mit ihnen zu machen (z. B. MyEtherWallet, MetaMask).

Wann und wie kann ich meine DCN-Token sehen?

Dentacoins werden automatisch an Ihr Wallet gesendet, sobald die Transaktion durchgegangen ist. Um Ihre DCN zu sehen, müssen Sie Dentacoin als benutzerdefiniertes Token in Ihrem Wallet hinzufügen:
Token Adresse: 0x08d32b0da63e2C3bcF8019c9c5d849d7a9d791e6
Token Symbol: DCN
Dezimalen: 0
Sie können Ihren DCN-Kontostand kontrollieren, indem Sie die folgende Website besuchen:
https://etherscan.io/token/0x08d32b0da63e2C3bcF8019c9c5d849d7a9d791e6?a=“enter-your-eth-address-without-quotes”

Kann ich meine ETH und DCN auf einer Austauschplattform speichern?

Technisch gesehen könnten Sie das. Aber wir raten Ihnen dringend dazu, das nicht zu tun. Das langfristige Speichern eines Wertes in einer Börse bedeutet, dass Sie keine Kontrolle über Ihren privaten Schlüssel haben, die Plattform hingegen schon.
Für die Speicherung von Dentacoin empfehlen wir Ihnen, eine Geldbörse zu verwenden, für die Sie Ihren privaten Schlüssel (z. B. MyEtherWallet, MetaMask) haben. Für ein höheres Sicherheitslevel können Sie ein Hardware-Wallet wie TREZOR oder Ledger Nano S in Betracht ziehen.

Wie kann ich meine Transaktions-ID finden?

Die Transaktions-ID wird meistens als TxHash bezeichnet.
Um Ihre Transaktions-ID zu sehen:
1) Gehen Sie auf ETH & ERC20: https://etherscan.io
2) Geben Sie die Adresse ein, von der Sie ETH gesendet haben, dann können Sie Ihre Transaktions-ID / TxHash einsehen.
