理念总结

Dentacoin 旨在全球范围内改进牙科护理，通过使用区块链优势让牙科护理变得价格实惠。我们相信使患者成为这个行业的活跃部分是打造牙科护理未来的关键。试点 Dentacoin 项目 (Trusted Review Platform) 作为第一个透明的、抗干扰和防操纵平台，已经在主网上运行。

✓ 将模式从治疗转变为预防。 通过把患者和牙医引入同一个社区，实现多种双赢效果。通过“希波克拉底”牙科保险模式，患者不再需要支付治疗费用，而是支付预防费用。 患者可以每月以最低的价格用 DCN 支付购买终生的、全面的牙齿保健方案。一份自动生效的智能保险协议将保证共同责任，无操纵以及低费用。通常有保险公司获取的利润将由患者和牙医共同分享。

✓ 通过公众的力量创造价值。 通过即将开发的基于区块链的工具（可信评论平台，一款诊后移动应用，一个教育网站），为基金会的使命做贡献的患者和牙医可以获得 Dentacoin 奖励。 Dentacoin 将创建一个患者的声音可以被听到的论坛，并帮助患者改善口腔健康习惯。同时，DCN 工具将允许牙医基于收集的反馈改进服务质量，同时也将围绕患者创建一个忠诚的患者社区——各服务业最重要的资产。

✓ 让牙科护理变得价格实惠。 使用收集到的 Dentacoin，患者可以: • 支付治疗费用; • 支付牙科保险; • 购买口腔护理产品。
对于牙医来说，Dentacoin 让他们可以: • 有稳定的基本收入; • 通过 Dentacoin 交易平台，购买更便宜的牙科材料和器械; • 创建患者社区; • 降低营销成本; • 降低治疗成本。

✓ 跨越牙科行业界线。 Dentacoin 还计划研发一种分散的、高度安全的患者健康记录数据库。健康数据将被高度保护、只有患者或医生可以通过特定许可进行更新和访问。 搜索健康记录时，数据库将简化患者和牙医的生活，并帮助提供最准确的诊断。

✓ 间接利润分享。 为了创造长期价值增长先决条件，Dentacoin 将执行一种所有 Dentacoin 持有者间接利润分享的独特方法。每年 Dentacoin 基金会将用上一个财务年度所得利润的 50% 购买 DCN，包括资产的权益、基金会收购的诊所的盈利和其它投资收益。因此，全球 DCN 供应减少；需求和市场价格将会随之增长。
